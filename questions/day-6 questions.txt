**28.expalin the difference between rowspan and colspan.
Answer)  Colspan attribute merges two or more columns into a single column. Rowspan attribute merges two or more rows into a single row.
--------------------------------------------------------------------------
**29.explain the diff bw cellpadding and cellspacing.
Answer)  Cellpadding basically defines the space present between a table cell's border and the content present in it. Cellspacing basically defines the space present between individual adjacent cells.

31.expalin all table elements and table attributes.
Answer)   The <tr> element defines a table row, the <th> element defines a table header, and the <td> element defines a table data/column.

	where as table attributes are:
border:specifies the width of the border around the table.
cellpadding: specifies the space between the content of a cell and its border.
cellspacing: specifies the space between the cells.
width:specifies the width of the table.
align:aligns the table within the containing elements.
valign:vertical alignment of content within cells.
summary:provides the discription or dummary of the table.
